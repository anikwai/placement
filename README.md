# Student Placement System

A secure platform for accessing Year 7 placement information.

## Tech Stack
- **Framework**: Laravel 10 + Inertia.js
- **Frontend**: React (TypeScript) + Shadcn UI + Tailwind CSS
- **Features**: Server-side pagination, instant search, responsive design.

## Usage
Run `npm run dev` and `php artisan serve` to start the development environment.
